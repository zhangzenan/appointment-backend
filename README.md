# 项目介绍

#### 本项目是预约服务小程序的后端服务

### 环境说明
* JDK 21
* SpringBoot 3.4.3
* 依赖管理工具Gradle

### 启动项目
1、执行doc文件夹中scripts.sql脚本，用户表中会创建默认用户手机号是13800138000，登录时需用这个手机号否则会登录失败

2、修改resources/application.yml文件，将数据库连接信息修改为自己的连接信息,其中包括数据库地址、数据库名、用户名、密码

3、运行AppointmentBackendApplication入口文件

### 项目所包含的功能
* 用户登录，会通过手机号校验用户是否存在，验证码是默认值123456
* 会通过JWT创建token和解析token
* 新增预约服务和查询预约服务列表
* 新增预约服务接口幂等性采用数据库唯一性验证，用户id、日期、时间段。当前采用简单的数据库约束，在实际项目中需要结合redis做前端幂等性校验，避免用户频繁点击提交按钮
